### Build Setup

``` bash
# install dependencies
npm install

# start server at localhost:4000
npm run start

# Compiles and hot-reloads for development
npm run serve

# Compiles and minifies for production
npm run build

# Run your tests
npm run test

# Lints and fixes files
npm run lint
```
